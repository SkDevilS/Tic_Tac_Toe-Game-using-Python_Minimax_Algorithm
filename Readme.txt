This is a simple AI based Tic Tac Toe game using Python Minimax and basic search algorithm.
the player against you is System
It uses 'X' and 'O' for playing the traditional Tic Tac Toe Game.
Since it is a very basic level code to understand Minimax and basic search algorithm, it cannot compete with the human player with game strategies.
